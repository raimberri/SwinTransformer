# SwinTransformer
SwinTransformer pytorch implementation for self-learning.   
## Environments
1. Ubuntu 20.04
2. Python 3.10
3. PyTorch 1.11
## Results
Model trained on 1 single GPU(2080 Ti)
|    | ACC@1 | ACC@5 |
|:-:|:-:|:-:|
|22 Epochs | 53.50%   |     78.14% |
